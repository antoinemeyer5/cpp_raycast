## Developer's zone

### Build & Launch

```zsh
$ git clone https://github.com/libsdl-org/SDL.git vendored/SDL
$ cmake -S . -B build   # setup
$ cmake --build build   # compile/build
$ ./build/bin/main      # execute
```
